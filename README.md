## 🛠 Troubleshooting

If skillless does not work as expected:

- Make sure your internet connection is active.

- Restart your computer and try opening the app again.

- Ensure you have enough disk space to download new skills.

- If installation fails, try running the installer as administrator. Right-click the file and choose **Run as administrator**.

- Check your antivirus or security settings. Some programs block installers; allow skillless through your firewall.

## 📂 Where files are stored

skillless saves data in these default locations:

- Program files: `C:\Program Files\skillless`

- User data and downloaded skills: `C:\Users\<YourUsername>\AppData\Local\skillless`

You can access these folders to back up or remove data manually if needed.
